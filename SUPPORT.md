# Support

Before reporting a problem, open **Settings → Diagnostics**, refresh provider status, and export the sanitized diagnostics report if it is relevant.

- Report reproducible bugs with the [bug report form](https://github.com/Aforno/Bartender/issues/new?template=bug_report.yml).
- Request product improvements with the [feature request form](https://github.com/Aforno/Bartender/issues/new?template=feature_request.yml).
- Browse existing reports in [GitHub Issues](https://github.com/Aforno/Bartender/issues).
- Report security vulnerabilities privately as described in [SECURITY.md](SECURITY.md); do not post credentials, prompts, generated source containing secrets, or personal diagnostics publicly.

Include the Bar Tender version, macOS version, Mac architecture, provider CLI/version when relevant, exact reproduction steps, and the smallest sanitized diagnostic excerpt needed to explain the failure.
